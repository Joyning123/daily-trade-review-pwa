# 执行约束卡

## 核心目标

帮助自己控制情绪、严格执行交易纪律，而不是预测行情。

## 强制风控规则

- 单笔仓位必须 `<= 10%`
- 每笔止损必须固定为 `-10%`
- 止盈规则必须固定为 `+30% 卖出一半`
- 禁止追涨加仓

## `1000u` 账户执行版本

如果你现在手里是 `1000u`，那这 4 条规则应当被翻译成下面这些具体动作：

- 单笔最大仓位 = `100u`
- 单笔止损触发后，理论亏损上限约 = `10u`
- 单笔到达 `+30%` 后，先卖一半，先锁定约 = `15u`
- 不允许因为 FOMO 在拉升中继续补仓，把一笔计划内交易变成失控暴露

这个换算只是在执行层面把百分比变成金额，不代表建议你重仓做高波动土狗。

## 盘前自检

出手前必须连续问自己 3 个问题：

1. 这笔交易是不是热点方向？
2. 这笔交易有没有成交量支持？
3. 这笔交易是不是在上涨趋势中入场？

只要有 1 个问题回答为“否”，默认不做。

## 违规处罚规则

为了让纪律真正产生约束，这里采用阶梯处罚：

- 当天出现 `1` 次违规：次日单笔仓位上限降到 `5%`，且最多只允许做 `2` 笔交易
- 当天出现 `2` 次违规：次日只允许做 `1` 笔试错单，单笔仓位仍然 `<= 5%`
- 当天出现 `3` 次违规，或出现任意一笔 `情绪交易 = 0分`：次日停止实盘，只允许复盘和观察
- 连续 `2` 天出现 `0分` 交易：强制休息 `48` 小时，并复盘最近 `10` 笔交易

## `1000u` 账户处罚换算

- 当天出现 `1` 次违规：次日单笔仓位上限降到 `50u`，且最多只允许做 `2` 笔交易
- 当天出现 `2` 次违规：次日只允许做 `1` 笔试错单，单笔仍然 `<= 50u`
- 当天出现 `3` 次违规，或出现任意一笔 `0分` 情绪交易：次日停止实盘

## 每日最低执行标准

- 没有设置止损，不允许下单
- 不因 FOMO 临时加仓
- 不因为上一笔亏损而立即“翻本单”
- 每笔交易必须能说明入场理由，否则视为情绪单
